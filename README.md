# ⚡ Cyber Chrono - Futuristic Timekeeping & Telemetry Suite

A modern, high-performance web dashboard for precision timekeeping, lap logging, and session telemetry. Designed with a sleek glassmorphism aesthetic, ambient radial lighting, dynamic animations, and multiple high-contrast cyberpunk/synthwave themes.

## ✨ Features
- 🎨 **Dynamic Themes:**
  - **Cyber Neon:** High-tech indigo and magenta neon aesthetics.
  - **Deep Space:** Dark emerald aurora with electric blue highlights.
  - **Arcade Synth:** Vibrant 80s retro-futuristic pink and orange glow.
- ⏱️ **Stopwatch & Lap Tracking:** Real-time millisecond or second precision with split delta tracking.
- ⏲️ **Countdown Timer:** Intuitive visual countdown with interactive status updates.
- 📊 **Local Telemetry & Analytics:**
  - Persists session statistics across page refreshes via `localStorage`.
  - Tracks total run cycles, total laps, and individual mode runtimes.
  - Keeps an active activity log for recent timer/stopwatch sessions.
- 📱 **Fully Responsive Layout:** Built with CSS Grid and Flexbox for seamless usage on desktop and mobile.

💾 Telemetry Data & Persistence
The application saves usage statistics to the browser's localStorage under the following keys:
timeSuite_Stats - Stores accumulated milliseconds, lap counts, run cycles, and activity logs.
timeSuite_Theme - Saves your current theme selection.
To clear all saved metrics, use the Flush Metrics button located inside the Telemetry panel.
